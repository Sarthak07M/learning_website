🚀 IET Study Hub
A centralized resource portal designed to streamline academic learning for engineering students at IET DAVV.
📖 Overview
IET Study Hub is a student-centric platform built to eliminate the chaos of finding quality study materials. By organizing resources through a hierarchical "Branch $\rightarrow$ Semester $\rightarrow$ Faculty" system, it provides instant access to Handwritten Notes, Previous Year Questions (PYQs), and Subject Assignments.
✨ Key Features
Dynamic Resource Navigation: A seamless "drill-down" UI that lets students filter content by branch, semester, and faculty.
Glassmorphism UI: Modern, dark-themed design with interactive ripple effects, custom cursors, and parallax animations.
Performance Optimized: Throttled scroll events and intersection observers for a smooth, lag-free experience.
Easter Eggs: Hidden Konami code functionality for an interactive developer touch.
Responsive Design: Optimized for mobile, tablet, and desktop viewing.
🛠️ Tech Stack
Frontend: HTML5, CSS3 (Custom Variables & Keyframes), Vanilla JavaScript (ES6+).
Architecture: State-driven dynamic rendering for hierarchical academic data.
Animations: Intersection Observer API & CSS Keyframe animations.
Deployment: Vercel / GitHub Pages.
